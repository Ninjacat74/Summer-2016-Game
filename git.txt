git clone <url> to clone git repository to current folder

git add <filename> or -A (all files)
git commit -m "<message>" to commit change to local repository
git push - to commit changes to remote repository.

git status - views status of files in directory


git fetch --all
git reset --hard origin/master


to allow git to overwrite your files:

 git add *
 git commit -a -m "local file server commit message"

 git fetch origin master
 git merge -s recursive -X theirs origin/master
